# Stackyon Coding Standards

## Angular 14
- NgModules only
- Syncfusion EJ2 for all UI
- HttpClient for API calls
- environment.ts for URLs
- takeUntil for Observables
- SCSS only, no inline styles

## .NET 8
- Three-layer: Controller > Service > Repository
- IHttpClientFactory for HTTP calls
- Dependency injection throughout
- xUnit tests 70% coverage minimum
